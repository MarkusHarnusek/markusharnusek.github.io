Tutoring Platform

Tutoring Platform ist eine Anwendung um sich selbst als Nachhilfelehrer anzubieten und zu präsentieren.

Interessenten können Informationen über den Nachhilfelehrer und die Fächer die dieser unterrichtet einsehen, den Nachhilfelehrer kontaktieren und Nachhilfestunden dierekt auf der Seite buchen.

Diese Anwendung verwende ich auch für mein Angebot von Nachhilfe an der HTL Leonding.

Das Projekt ist noch immer in Entwicklung.

TCPChat

TCPChat ist ein einfaches konsolenbasiertes Chat-Programm, dass das TCP-Protokoll verwendet.

Es besteht aus einem Server und einem Client. Der Server kann mehrere Clients gleichzeitig bedienen.

Server ports können durch die Bearbeitung des Quellcodes angepasst werden.

Die Clients können sich zu einem beliebigen Port verbinden, sich einen Namen geben und Nachrichten versenden.

Eine Ansicht zweier TCPChat Clients und eines Servers auf dem Port 8080

Technische Details

  • Programmiersprache: C
  • Netzwerkprotokoll: TCP
Link zur GitHub Repository

USRTG

Unified Sim Racing Telemetry Getter

USRTG ist ein Programm zum Abrufen und Anzeigen von Telemetriedaten aus verschiedenen Rennsimulationen.

Es unterstützt Spiele wie Assetto Corsa

Weitere Spiele werden in Zukunft unterstützt.

Applikationen wie Racing Dashboard können mit USRTG verbunden werden um die Telemetriedaten in Echtzeit anzuzeigen.

Technische Details

  • Programmiersprache: JavaScript
  • UI-Framework: CSS
