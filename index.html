AeroMesh Drones

Erstellung 3D-Höhenmodelle aus Luftbildaufnahmen

Umfangreiche Dokumentation zum eigenständigen Bau von Drohnen, zur Datenerfassung und zur automatisierten Generierung von 3D-Höhenkarten mit der bereitgestellten Software. Die Anleitung enthält Berechnungen, Software-Tools, detaillierte Teilelisten, Schritt-für-Schritt-Aufbauten sowie KI-gestützte Verarbeitungs- und Analyseverfahren.

Die Inhalte basieren auf einer modularen, eigenen Dokumentationsvorlage mit Seitenerstellung mithilfe von MarkDown Dateien.

TCPChat

TCPChat ist ein einfaches konsolenbasiertes Chat-Programm, dass das TCP-Protokoll verwendet.

Es besteht aus einem Server und einem Client. Der Server kann mehrere Clients gleichzeitig bedienen.

Server ports können durch die Bearbeitung des Quellcodes angepasst werden.

Die Clients können sich zu einem beliebigen Port verbinden, sich einen Namen geben und Nachrichten versenden.

Eine Ansicht zweier TCPChat Clients und eines Servers auf dem Port 8080

Technische Details

  • Programmiersprache: C
  • Netzwerkprotokoll: TCP
Link zur GitHub Repository

USRTG

Unified Sim Racing Telemetry Getter

USRTG ist ein Programm zum Abrufen und Anzeigen von Telemetriedaten aus verschiedenen Rennsimulationen wie Assetto Corsa. Weitere Spiele werden in Zukunft unterstützt.

Applikationen wie Racing Dashboard können mit USRTG verbunden werden um die Telemetriedaten in Echtzeit anzuzeigen.

Technische Details

  • Programmiersprache: JavaScript
